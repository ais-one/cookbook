# REST Example

> A REST example project using vue-crud-x component


## Quickstart

**Pre-requisites**

Please install REST backend using the README.MD in the "backend" project folder

1. Rename .env-sample file to .env.development

2. Install packages

```
npm i
```

3. Start The Client & Server

```
npm run start
```
