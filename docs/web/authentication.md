## Support

The following are supported
- access_token and refresh_token handling by self-made auth server
  - password login + OTP
  - SAML2
  - OAuth2
- access_token and refresh_token handling by certified auth servers (using keycloak in our case)
  - OpenID Connect (OIDC)
    - grant types are
      - authorization_code
      - refresh_token
